◆ GetPercentage()

float Timer::GetPercentage ( )

获取定时器完成百分比

返回
[0.0f, 100.0f]数据为当前定时器完成百分比

获取定时器完成进度float数据类型百分比

注意
计算顺序为若设定时间小于等于0则始终为100，若当前时间小于等于0始终为0，若两者皆否才正常计算百分比

◆ Pause()

void Timer::Pause ( )

暂停定时器

定时器暂停，即使调用 Update(int delta) 不会执行任何工作

注解
此操作不会变更标志，定时器停止计时以更新状态

◆ ReSet()

void Timer::ReSet ( )

重启定时器

重置定时器将设定定时器为初始状态

注解
此操作将会改变是否到达设定时长标志为false并将当前时间重置为0并暂停计时器，需手动启动定时器

◆ SetAutoRestart()

void Timer::SetAutoRestart ( bool is_auto)

设定定时器是否自动重启

参数
is_autotrue自动重启，false手动重启

自动重启通常用作循环触发回调函数，启动则重复执行，关闭则触发后需手动执行 ReStart() 重启标记

◆ Update()

void Timer::Update ( int delta)

更新定时器状态

参数
delta距离上次调用间隔时长

传入上次调用定时器更新状态间隔时长以更新定时器状态

注解
函数不会跳过任何数据，若定时器运行中即使负数时长也会进行计时
注意
传入间隔时长需要确保所使用的单位一致，函数不进行单位转换
